**Step 1: Clone the Repository**

**Step 2: Make the script executable**
```bash
chmod +x toolkit.sh
```

**Step 3: Run the Toolkit**
```bash
./toolkit.sh
```

- Enter the number corresponding to the tool.
- Follow the on-screen instructions.
